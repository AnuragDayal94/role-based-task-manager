# Role-Based Task Management System (MERN)

A full-stack MERN application with authentication, authorization, and role-based task management.

---

## Features
- JWT-based authentication
- Role-based access control (Admin/User)
- Secure password hashing
- Task creation, assignment, update, and deletion
- Backend-enforced authorization

---

## How Authentication Works

1. User logs in with email and password
2. Backend verifies credentials and returns JWT
3. Token is stored in frontend
4. Token is sent in Authorization headers
5. Backend validates token and role for every protected request
